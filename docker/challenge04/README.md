# Challenge 04

 En este desafío se centra en automatizar la construcción, pruebas, empaquetado y despliegue de las aplicaciones Java y Go, junto con la base de datos PostgreSQL, utilizando Docker y Docker Compose. . Incorporar prácticas de versionado semántico y asegurar una integración continua fluida para garantizar la entrega continua y la estabilidad del sistema."


## Ejecucion

Ejecutar el siguiente script [challenge04.sh](/docker/challenge04/challenge04.sh) y se desplegara el proyecto.

Considerar dar permiso al script para ejecutarlo:
    
    chmod +x challenge04.sh
